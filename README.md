# ADA_F
Proyecto final de Análisis y Diseño de Algoritmos

### Como correrlo:
1. Abrir la terminal en VS code (Ctrl + Shift + M)
2. En la carpeta raíz, escribir "go run main.go"
3. Click en Aceptar en caso salga un aviso de permiso para main
4. Ctrl + Click en el link de localhost que aparecerá en consola para abrir la página del navegador
 
*Nota: Se necesita tener lenguaje Golang instalado en vs code*
